chembl_al-packge
chembl_al is a lightweight Python package designed to simplify and accelerate data extraction from the ChEMBL database, eliminating the need for repetitive and complex API coding.

Project Objective

The goal of chembl_al is to provide a clean and unified interface for researchers in computational chemistry, medicinal chemistry, and machine learning to access ChEMBL data (compounds, bioactivity, targets, and more) in just a few simple steps.

- Key Features
Easy retrieval of data from ChEMBL with minimal code
Reduces the complexity of working with the ChEMBL API
Outputs data in analysis-ready format (Pandas DataFrame compatible)
Designed for:
QSAR modeling
Molecular docking preprocessing
Drug discovery pipelines
Machine learning dataset generation

- Usage

The package abstracts API handling and data cleaning into simple functions, allowing users to focus on analysis and model building rather than data engineering.

- Research Applications

chembl_al is suitable for:

Computational drug design
Cheminformatics workflows
Bioactivity prediction models
AI-driven drug discovery research

- Core Idea

Instead of writing long scripts to query and process ChEMBL data, chembl_al lets you do it in just a few lines of code, making data access fast, consistent, and research-friendly.
